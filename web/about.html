Why ILMA?

Words Children Know

Every keyword — say, remember, recipe, blueprint — is a word a child already understands. No jargon.

Knowledge Built In

Modules teach finance, science, health, and Islamic values. Coding lessons that also teach how to live.

Compiles to C

Native performance. Runs on any device ever made — from a Raspberry Pi to a supercomputer.

Grows With You

Three tiers: Seed (ages 3-7), Sapling (7-13), Tree (13+). One language, three depth levels.

Arabic & Urdu Support

Full Unicode from day one. A child in Karachi or Cairo can write code in their own script.

Free Forever

The compiler, web IDE, and all 30 lessons are free. A child with a phone and wifi can learn ILMA.

The Manifesto

"What if we started with a child? What if the syntax was invented fresh, borrowing nothing, using only the words a child already knows?"

Every programming language that came before ILMA was designed by and for adult software engineers. Python simplified C. Ruby simplified Python. But none of them asked the fundamental question.

ILMA breaks this chain entirely. Every keyword is chosen by asking: if a child who has never seen code before reads this line, will they understand what it does?

The standard library is not just code — it's a curriculum. When a child calls finance.zakat(), they're not just learning function calls — they're learning about wealth, responsibility, and the mathematics that connects them.

ILMA's name comes from the Arabic word ilm — knowledge, science, wisdom. The first divine command was Iqra — Read. We carry that mission: give every child the tools of the knowledge economy, regardless of geography, wealth, or school system.
