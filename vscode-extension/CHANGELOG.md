# ILMA Language Extension — Changelog

## 0.1.0

- Initial release
- Syntax highlighting for all ILMA keywords including multi-word keywords
- 10 code snippets (hello, recipe, blueprint, if, repeat, foreach, while, try, use-finance, use-time)
- Run ILMA programs with Ctrl+F5 / Cmd+F5
- Status bar run button
- Right-click context menu "Run ILMA File"
- `.ilma` file association
